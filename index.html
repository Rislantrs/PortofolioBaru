Telco Churn

Proyek Telco Customer Churn Prediction ini membangun model klasifikasi churn berbasis XGBoost untuk membantu tim retensi mengidentifikasi pelanggan berisiko tinggi secara proaktif. Menggunakan dataset IBM Telco (7.043 pelanggan), pipeline mencakup data cleaning, EDA, preprocessing (scaling, encoding, imputasi), penyeimbangan kelas dengan SMOTE, dan tuning hyperparameter via GridSearchCV. Model akhir mencapai F1-Score 92.1%, Recall 89.8%, Precision 94.4%, dan ROC AUC 99.0%, dengan interpretasi fitur menggunakan SHAP. Output mencakup model siap deploy (`best_xgboost_model.pkl`) dan analisis prediksi pelanggan (`xgboost_predictions_analysis.csv`). Tools utama: pandas, numpy, scikit-learn, imblearn, xgboost, joblib.
